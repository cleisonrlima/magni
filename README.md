# magni

Exemplo simples de um projeto .NET c/ Angular.

# backend:
  Para rodar o backend: </br>
   $ cd backend/ </br>
   $ dotnet ef  database update (esse comeando vai gerar o banco local) </br>
   $ dotnet run </br> </br>
  
 # frontend:
 Para rodar o backend: </br>
  $ cd frontend/ </br>
  $ npm install </br>
  $ ng serve </br>
